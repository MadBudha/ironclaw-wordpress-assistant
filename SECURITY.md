# Security Policy

We take security seriously. If you find a vulnerability, please report it responsibly.

## Reporting a Vulnerability
1. Email us at: kcameau@gmail.com

2. Include:
   - A clear title
   - Steps to reproduce
   - Impact description
   - Suggested fix (if known)

3. We’ll respond within 48 hours.

## What We Don’t Accept
- Public disclosures without prior notice
- Exploits or malicious intent

## Responsible Disclosure
We’ll acknowledge your report and work to fix it quickly. No action will be taken against you.
